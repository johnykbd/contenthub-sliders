# Sitecore Content Hub External Component Examples

This repository provides some examples on how to write and build a JavaScript ECMAScript module as an external component for Content Hub.

In order to build the examples, you need to first install the dependencies by running ```npm install ``` and then proceed with building by running ```npm run build```. The generated bundled code will be made available to you in the dist folder located in the corresponding example folder.

JavaScript SDK:

- [@sitecore/sc-contenthub-webclient-sdk](https://www.npmjs.com/package/@sitecore/sc-contenthub-webclient-sdk)
